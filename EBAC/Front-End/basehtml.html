Aula 1:

Introdução:

O head se baseia na parte não visual mas informativa do HTML onde possui informações de titulo tag(title), tipo de caractere tag(meta charset ="UTF-8"), etc.
Já o body armazena o visual do HTML onde existem diversas tags para modificar o visual de uma parte.

Headers:

A tag h(numero) serve para poder criar tamanhos para cabeçalhos onde vai desde o 1 até o 6:

h1

h2

h3

h4

h5
h6

Span e P:

Servem para separar partes do texto onde não são devidamente aparentes, a tag "p" define um paragrafo de texto já a tag span separa uma parte do texto para ser alterada ou estilizada

Bold, Italic e Mark:

A tag "b" serve para deixar os textos em negrito, enquanto a tag "i" tranforma o texto em italico e por fim a tag "mark" serve para destacar os textos como se fosse um marca-texto. Uma propriedade é que as tags podem ser usadas consecutivamente formando um possivel negrito e italico, negrito e marcado e italico e marcado

Tags a, em, strong

tag "a": utilizada para construir links, informamos para qual endereço o usuário será levado através do atributo href
tag "em": utilizada para enfatizar um texto, também aplica o itálico
tag "strong": para informar que um texto é importante, também aplica o negrito

tags del, small, blockquote e address

<del>: utilizada para informar que um texto foi deletado, aplica o tachadono texto
<small>: utilizada para criar um texto com fonte menor
<blockquote>:
utilizada para inserir uma citação, aplica um margem no conteúdo

<address>:
utilizada para inserir um endereço, também aplica o itálico
