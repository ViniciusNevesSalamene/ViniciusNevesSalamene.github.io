Div

As divs são a base para divisão de conteudos e estilos para um site onde anteriormente elas eram as unicas existentes mas agora no HTML existem outros tipos de containers

Header

A tag <header> serve como uma divisão semantica para o cabeçalho do site onde especificamente terá apenas este conteudo referente ao cabeçalho.
Navigator
Uma categoria muito usada no header que define uma barra de navegação entre paginas
Inicio
Containers
Ir para o containers
Ir para o tabelas

Main

A tag <main> serve para manter seu conteudo principal do site podendo ser distribuidos em subtags chamadas <section> onde dentro desta tag podemos incluir mais uma subtag chamada de <article>

Aside

Esta tag se torna bastante útil no quesito de criar uma aba lateral para o site. Deixando em destaque que todas essas tags se baseiam novamente em semantica apenas, toda a estilização será feita pelo arquivo CSS.

Footer

A tag <footer> serve para definir o rodapé de uma página como por exemplo contato com o dono da página ou até mesmo o destaque de direitos autorais, podendo até mesmo ser um menu inferior.
